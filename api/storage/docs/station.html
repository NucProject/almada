1. 自动站列表 

API

GET /d/stations

注释

 列出当前用户组下所有的自动站

返回值字段

名称

类型及范围

说明

其他

list.0.stationId

list.0.stationName

string

自动站名称

list.0.stationDesc

string

自动站描述

list.0.stationAddress

string

自动站地址

list.0.stationType

int

自动站类型

list.0.stationSn

string

Station SN(目前还没有支持)

返回值示例(data)

{
    "list": [
        {
            "stationAddress": "房山XX-XX",
            "stationDesc": "北京全自动站(房山)",
            "stationId": null,
            "stationName": "北京全自动站",
            "stationSn": "S00001",
            "stationType": 1
        }
    ]
}

2. 创建自动站 

API

POST /d/station

注释

 在当前用户组下创建一个自动站

表单参数

名称

类型及范围

说明

必选

stationName

string

自动站名称

Yes

stationDesc

string

自动站描述

Yes

stationAddress

string

自动站地址

Yes

stationType

int

自动站类型 (1, 固定自动站; 2, 移动自动站; 3, 虚拟自动站)

Yes

返回值字段

名称

类型及范围

说明

其他

stationName

string

自动站名称

stationDesc

string

自动站描述

stationAddress

string

自动站地址

stationType

int

自动站类型

调用示例

curl -d"stationName=station&stationDesc=XYZ&stationAddress=someAddress&stationType=1" http://127.0.0.1:1024/d/station?groupId=1

返回值示例(data)

{
    "stationAddress": "房山XX-XX",
    "stationDesc": "北京全自动站(房山)",
    "stationName": "北京全自动站",
    "stationType": 1
}
